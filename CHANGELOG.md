# Changelog

All notable changes to MayTTK will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-30

### Added
- Initial release
- Real-time Time To Kill (TTK) calculation
- Adaptive DPS smoothing algorithm for accurate estimations
- Trend indicators showing TTK direction (^ up, v down)
- Configurable display frame with drag-to-position
- Attach-to-target-frame mode with offset controls
- Auto-detection for ElvUI and Shadowed Unit Frames
- 6 color presets (Classic, Warm, Cool, White, Gold, Purple)
- 7 built-in font options
- Minimap button using LibDBIcon (compatible with all minimap shapes and LDB displays)
- Addon Compartment support (Retail 10.x+)
- Full localization support with 9 languages (EN, DE, FR, ES, PT, RU, zhCN, zhTW, KO)
- Modular locale structure (Locales folder with per-language files)
- Slash commands for quick control
- Frame locking and position reset
- Preview mode for easy positioning
- Background and border customization
- Configurable opacity and scale
- Combat log damage and absorb tracking
- Automatic data clearing on combat end
- Cross-version compatibility (Retail, Classic Era, Cata Classic)
